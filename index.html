🌟①基本

LINEの目的

  • クライアントさんの目標を達成するためのコミュニケーションツール

お客様は感情を表に出さずに話すことが多い。私たちは、相手の感情を読み取ることをLINE上で汲み取らないといけない。

～LINE返信文の作り方～（相手から不安の自己開示があったとき）

  1. 感謝
  2. 共感（心理的な安全性の確保）→いきなり励ますのはNG
  3. 承認（自己開示への感謝）／IMessage（自分の気持ちを伝える）のもOK
  4. ヒアリング　❌すぐ解決策は出さない

LINE独り立ちするまで

  1. LINE FBシートに記入する
  2. 複数できたらSlackでリーダーをメンションして「LINE FBシートの確認お願いします」と伝える
  3. リーダーからのチェックがついたら送信
  4. 送信済みにチェックする

②面談調整

必ず何の面談か、誰との面談かチェックする！

🌟コーチとの面談の場合

  1. こちらから日程を提示する場合 → そのコーチの予定を見て日程を提案 → その旨をSlackで担当コーチに報告
  2. お客さん側から日程を提示してくれた場合 → そのコーチの予定を見てLINE返信 → カレンダー招待 → その旨をSlackで担当コーチに報告
  3. 日程が確定した場合 → 面談を行うコーチの面談URLをお客様に送付（コーチ別の面談URLは⑥コーチ番号・面談URL・日程調整URLのページに貼ってあります！）
カレンダー招待のやり方

自分のカレンダーに「〇〇さん」という予定を指定の時間に入力し、担当コーチを招待 → 自分は「参加しない」にする

🌟SFAの飛ばし方

  1. Sales Force → 検索からお客様の名前をコピペして検索
  2. お客様の名前を押す（同姓同名だったらWH本気プランの人）
  3. カスタマーサポートのところを押す
  4. 右側の緑色のところを押す
  5. Todoのところで依頼分類は「LINE返信」
  6. 依頼内容はお客様の名前
  7. 割当先をToDoを飛ばしたい方や部署に変更
  8. メール件名のところに「LINE返信お願いします」と入力
  9. コメントのところにお客様からの返信をコピペ
  10. 保存 → １次返信（テンプレは下にあります）

💌⑤リテンション

リテンションとは、お客様のモチベーションを維持し、学習を継続してもらうために私たちコーチがLINEを送り、コミュニケーションをとることです。
  • 3日に一度ペアでお客様に返信できているか確認
  • これまでの人のリテンションを見て温度感を合わせる
  • LINEノートにリテンションで意識することを書いてくれてある時もあるから必ずチェック！
  • 「こんにちは！Studyln英語コーチです！」からLINEの文章を始める

リテンションの文章は以下のシステムを使ってもいい：
リテンションLINE 作成・添削システム

そのお客さんの学習進捗を確認してからリテンションを送るととても良い。ここから学習進捗を確認できる：
Coaching Interface
この学習ログから Command＋F でお客様の名前を検索 → お客様の名前を押す

全然連絡返してくれない人はリテンションしない方がいい（しっかり文面を見てコーチと相談する）
